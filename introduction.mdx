## Key Features

### AI Integration
Connect to leading AI providers with built-in support for conversation memory and tool calling.

- **OpenAI** - GPT-4o, GPT-4 Turbo, GPT-3.5
- **Anthropic** - Claude 3.5 Sonnet, Claude 3 Opus
- **Google** - Gemini 1.5 Pro, Gemini 1.5 Flash

### WhatsApp Automation
Send and receive WhatsApp messages with event-driven triggers.

- QR code pairing
- Message filtering by sender, group, or keywords
- Media support

### Android Device Control
Automate Android devices via ADB or remote WebSocket relay.

- 17 service nodes (battery, WiFi, Bluetooth, camera, etc.)
- App launching and system monitoring
- Sensor data access

### Webhooks & HTTP
Receive incoming webhooks and make HTTP requests to external APIs.

- Dynamic webhook endpoints
- Full HTTP method support
- Custom response handling

## Architecture Overview

MachinaOs uses a modern tech stack:

| Layer | Technology |
|-------|------------|
| Frontend | React 19, TypeScript, React Flow, Zustand |
| Backend | Python FastAPI, SQLAlchemy, LangChain |
| Real-time | WebSocket (25 message types) |
| Database | SQLite (dev), Redis (production cache) |
| Deployment | Docker Compose, nginx reverse proxy |

<Info>
MachinaOs is designed for self-hosting. Your data stays on your infrastructure.
</Info>
